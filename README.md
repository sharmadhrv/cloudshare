# 🚀 CloudShare

CloudShare is a secure and scalable file sharing platform built with **Java Spring Boot**. It enables users to register, log in using **JWT Authentication**, and manage files securely through modern REST APIs.

Designed as a real-world backend project inspired by platforms like **Google Drive** and **Dropbox**, CloudShare demonstrates backend development, security, database integration, and cloud deployment skills.

---

## ✨ Features

### 🔐 Authentication

* User Registration
* Secure Login
* JWT Token Based Authorization
* BCrypt Password Encryption

### 📁 File Management

* Upload Files
* View My Files
* Download Files
* Delete Files
* Share Files Securely

### ☁️ Cloud Ready

* Local File Storage (Current)
* AWS S3 Integration (Planned)
* AWS EC2 Deployment (Planned)
